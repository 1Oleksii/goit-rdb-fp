# Фінальний проєкт RDB

Виконано таки 5 кроків відповідно до завдання.

### 1. Завантаження даних
- Створено схему `pandemic`
- Імпортовано дані з файлу `infectious_cases.csv` через Table Data Import Wizard
- Усі числові колонки (`Number_*`) імпортовано як `TEXT` (щоб уникнути помилок з порожніми значеннями)
- Кількість рядків у таблиці: **10521**

Скріншот: `Крок1_схема_pandemic_імпорт_10521.png`

### 2. Нормалізація до 3НФ
- Створено таблицю `entities` з унікальними комбінаціями `Entity` та `Code`
- Створено таблицю `cases` з посиланням на `entity_id`, роком та числовими показниками
- Дані перенесено з оригінальної таблиці до нормалізованої структури

Скріншот: `Крок2_нормалізовані_таблиці_10521.png`

### 3. Аналіз Number_rabies
- Пораховано середнє, мінімум, максимум і суму для `Number_rabies` по кожній унікальній комбінації `Entity` + `Code`
- Порожні значення відфільтровано
- Результат відсортований за середнім у спадному порядку, виведено топ-10 рядків

Скріншот: `Крок3_аналіз_Number_rabies_top10.png`

### 4. Різниця в роках
- З колонки `Year` створено дату 1 січня відповідного року (`STR_TO_DATE`)
- Додано поточну дату (`CURDATE`)
- Вираховано різницю в роках (`TIMESTAMPDIFF(YEAR, ...)`)

Скріншот: `Крок4_різниця_років.png`
